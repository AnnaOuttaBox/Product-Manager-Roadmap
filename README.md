# Product-Manager-Roadmap
Este repositório reúne meu roadmap completo de transição para Product Management, incluindo estudos, práticas, ferramentas e um projeto aplicado que integra Inteligência Artificial, Neuromarketing, Neurociências e Psicologia Positiva.
O objetivo é demonstrar, na prática, as habilidades essenciais de um(a) Product Manager, desde a descoberta do problema até a entrega de um protótipo navegável e um estudo de caso final.

🎯 Objetivo do Projeto
Criar um produto digital capaz de recomendar conteúdos, produtos ou experiências personalizadas com base no comportamento do usuário, aplicando:
- IA para personalização
- Neuromarketing para entender gatilhos de decisão
- Neurociência para mapear atenção, memória e motivação
- Psicologia positiva para promover engajamento saudável

📅 Roadmap de Desenvolvimento (Resumo)

- Entrevistas com usuários

- Definição do problema
- Personas e jornadas
  
2. Definição
- User stories
- Critérios de aceitação

4. Construção do Estudo de Caso

5. Publicação

- Protótipo
- Estudo de caso
